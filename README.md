# SnuggleCraft Modpack

Official modpack for the SnuggleSMP modded server.

This pack is maintained using **Packwiz**, allowing contributors to safely add and update mods while keeping the pack reproducible.

---

## Requirements

Minecraft version: 1.21  
Mod loader: NeoForge / Fabric (depending on your pack)

---

## Installing the Modpack

### Option 1 — CurseForge Export
Download the latest release and import it into the CurseForge launcher.

### Option 2 — Packwiz Installer
Using Prism Launcher or a Packwiz installer:
packwiz-installer-bootstrap.jar <pack-url>
